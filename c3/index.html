Hello Servlet

Schildt, Herbert. Java: the complete reference. McGraw-Hill Education Group, 2014.

Steps:

1. Compile:

javac HelloServlet.java -classpath "<path_to_apache>/lib/servlet-api.jar"

2. Add *.class file(s) to tomcat

Copy .class file for example into 

/webapps/examples/WEB-INF/classes

3. Add servlet name and mapping

  1. Open 
    WEB-INF/web.xml
  2. Add servlet name: 
    <servlet>
<servlet-name>HelloServlet</servlet-name>
<servlet-class>HelloServlet</servlet-class>
</servlet>
