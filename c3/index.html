Hello Servlet

Schildt, Herbert. Java: the complete reference. McGraw-Hill Education Group, 2014.

Steps:

1. Compile:

javac HelloServlet.java -classpath "<path_to_apache>/lib/servlet-api.jar"

2. Add *.class file(s) to tomcat

Copy .class file for example into 

/webapps/examples/WEB-INF/classes

3. Add servlet name and mapping

  1. Open 
    WEB-INF/web.xml
  2. Add servlet name: 
    <servlet>
    <servlet-name>HelloServlet</servlet-name>
    <servlet-class>HelloServlet</servlet-class>
</servlet>
  3. Add Servlet mapping: 
    <servlet-mapping>
    <servlet-name>HelloServlet</servlet-name>
    <url-pattern>/servlet/HelloServlet</url-pattern>
</servlet-mapping>

Start Tomcat

Types of http requests

Hypertext Transfer Protocol: Communications between clients and servers

GET:

Requests data from a specified resource

demo?name1=value1&name2=value2

Can be used for forms too but NEVER for sensitive data

Should be used only to retreive data

POST:

Submits data to be processed to a specified resource

Data goes in http message body (test)

Method Description
HEAD Same as GET but returns only HTTP headers and no document body
PUT Uploads a representation of the specified URI
DELETE Deletes the specified resource
OPTIONS Returns the HTTP methods that the server supports
CONNECT Converts the request connection to a transparent TCP/IP tunnel
