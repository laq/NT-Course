The DOM

Document Object Model (DOM) is a programming interface for HTML, XML and SVG documents

Structured representation of the document as a tree.

Methods that allow access and changes to the tree, structure, style and content.

Document as a group of nodes and objects, possessing various properties and methods.

Nodes can also have event handlers attached

Essentially, it connects web pages to scripts or programming languages.

Although the DOM is often accessed using JavaScript, it is not a part of the JavaScript language.

It can also be accessed by other languages.
